# Installation

## Homebrew
```
    /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## Setup Node
```
    brew install node zsh zsh-completions openssl
    npm install -g yo babel-cli gitbook-cli babel-eslint eslint 
    npm install
```

## Dev Server
```
    npm run dev
```

## Production build
```
    npm run build
```
